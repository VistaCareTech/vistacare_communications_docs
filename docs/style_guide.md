# Contributing

If you want to contribute to the following repository you should follow the next sections to create a markdown document. Below you will find some useful links that allow you to create markdown documents.

* [Markdown Oficial documentation](https://www.markdownguide.org/cheat-sheet/)
* [Markdown Online viewer](https://markdownlivepreview.com/)
* [Screen record tool](https://obsproject.com/)
* [Video Editing tool](https://shotcut.org/)

The markdown documents can be created online or on a desktop platform.

To create a markdown on a desktop platform, you just have to follow the next steps:

1. Open a text editor.
2. Write some lines as the following in your text editor.

```
# Title

## Subtitle

This is a testing example for the documentation.

## Lists

### Unordered

* Item 1
* Item 2

### Ordered

1. Item 1
2. Item 2
```

3. Save the file with the extension .md
4. Test the result in the following link: [Markdown Online viewer](https://markdownlivepreview.com/)

## Style guide

The language style that we use is Markdown, if you have a question that can't be solved in this section, please refer to the [Markdown oficial documentation](https://www.markdownguide.org/cheat-sheet/).

1. The Titles of each section require the symbol `#`. This ensures that the title appears in the table of contents. For instance: 
    ```
    # contributing
    ```

2. The Subtitles of each section require the symbol `##`. For instance: 
    ```
    ## Style guide
    ```
3. To Highlight information you should use the symbol ` at the start and the end of the sentence.
For instance: 
    ```
    `##`
    ```

4. The Highlight Box allows you to add relevant information to the user in different ways:

* Warning

```
<div class="warning">
<p class="admonition-title">WARNING</p>
<p>Write your message here.</p>
</div>
```

<div class="warning">
<p class="admonition-title">WARNING</p>
<p>Write your message here.</p>
</div>

* Important 

```
<div class="note">
<p class="admonition-title">IMPORTANT</p>
<p>Write your message here.</p>
</div>
```

<div class="note">
<p class="admonition-title">IMPORTANT</p>
<p>Write your message here.</p>
</div>

* Tip

```
<div class="seealso">
<p class="admonition-title">TIP</p>
<p>Write your message here.</p>
</div>
```

<div class="seealso">
<p class="admonition-title">TIP</p>
<p>Write your message here.</p>
</div>

5. To emphasize a sentence, you have different ways to do it.

* `code style`
* *Italic style*
* **Bold style**
* ***Italic and Bold style***

6. The structure to create an order list is the following:

```
1. First item.
2. Second item.
3. Third item.
```

7. The structure to create an unorder list is the following:

```
* First item.
* Second item.
* Third item.
```
